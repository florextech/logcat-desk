# Roadmap

## Near term

- Better package-aware filtering by resolving app PID mappings through ADB
- Saved filter presets and per-project sessions
- Improved console ergonomics for long-running sessions
- More complete macOS release hardening, including signing and notarization

## Next

- Multi-device tabs
- Bookmarks and saved log snippets
- Disk-backed capture for long sessions
- Structured export formats beyond plain text

## Future

- App-level onboarding and diagnostics for adb setup
- Optional crash pattern detection and smart highlights
- Performance profiling for very high-volume log streams
